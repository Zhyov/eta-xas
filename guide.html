Alphabet & Phonology

Alphabet:
The e'ta xas alphabet has 46 letters being those 37 consonants, 8 vowels and 1 click sound.

The alphabet is as follows:
a, a', p, b, B, V, e, e', s, S, c, z, z', Z, t, O, T, t', d, d', D, f, v, h, x, i, i', j, J, k, g, y, l, y', m, m', n, N, n', o, r, r', R, u, O' and -

e'ta xas does not have an uppercase version for its letters, thus, there is no need to capitalize the letter at the start because it may end up referring to another sound.

The format of silables in e'ta xas is (C)V(C/KV) for the first silable and CV(C/KV) for the rest. The symbols are C = Consonant, V = Vowel, K = Click and (◌) = Optional.

In e'ta xas, the first silable is always stressed.

Grammar

Grammar Symbols:
  • ◌- = prefix
  • -◌ = suffix
  • ~◌ = suffix replacing the last letter of the word (only if the letter is a vocal)
Grammar Rules:
  • i- = past tense
  • a- = present tense
  • u- = future tense
  • o- = no verb tense (In e'ta xas it is grammatically correct to not use time for verbs)
  • -a = feminine singular
  • -o = masculine singular
  • -e = neutral singular
  • ~i' = masculine/feminine plural
  • ~e' = neutral plural
  • ◌~◌ = word connector (Example: In f~r'a'n [my name] the f~r' is only one consonant instead of two. You can write stuff like y~la~in and it would still follow the silable format rule. Only consonant + consonant and vocal + vocal is allowed for this.)
